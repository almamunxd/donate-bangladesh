What is the document object model

The Document Object Model (DOM) is a programming interface that represents a web page's structure and allows programs to change its content, style, and structure. The DOM is used for HTML and XML documents.

How do you select an element from the DOM?

To select an element from the Document Object Model (DOM) in JavaScript, you use methods like getElementById() to access an element by its unique ID, getElementsByClassName() to find elements with a specific class name, or querySelector() to select an element based on a CSS selector, which allows for more complex targeting using combinations of IDs, classes, and tag names.

What is event delegation in the context of the DOM, and why is it useful?

In the context of the DOM (Document Object Model), event delegation is a technique where you attach a single event listener to a parent element, allowing it to handle events triggered on any of its child elements, effectively "delegating" event handling to a higher level in the DOM tree;

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM in JavaScript, you can access the element using methods like getElementById, getElementsByClassName, or querySelector, then use the setAttribute method to change attributes and the style property to modify CSS styles directly on the element.
